# MyBike 📝

This is a landing page for the MyBike bicycle brand, created using HTML, CSS and SASS.

[Click here to see DEMO LINK](https://sholudyvyy.github.io/mybike-landing-page/)

## 📌 Features.

- Responsive design: The page adapts to different screen sizes and devices.
- Bicycle comparison: The page features three different bike models that can be compared.
- Contact form: There is a contact form on the page where users can leave their contact information.
- Stylish design: The page has a minimalistic but stylish design with a dark color scheme.

## 🛠️ Technologies

- **HTML5**
- **CSS3**
- **SASS**
- **BEM**

## 🚀 How to launch?

   ```bash
   git clone https://github.com/Sholudyvyy/mybike-landing-page.git
   cd mybike-landing-page
